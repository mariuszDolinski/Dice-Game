# Dice-Game
Its a desktop app which allows to play Dice Game. Max number of players: 4 (on the same computer).
Interface language: polish

<h1>Technology used</h1>
<ul>
<li>Microsoft Visual Studio (ver. 10 to 2022)</li>
<li>Windows Forms</li>
<li>C#</li>
</ul>
